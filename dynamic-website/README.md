# dynamic-website

This is a full-stack web application using Node.js (backend), React (frontend), Supabase for authentication, Stripe for payments, and a CLI for generating quotes. The project is structured for deployment on Render, with GitHub as the repository host.

## Structure
- `backend/` — Node.js API server
- `frontend/` — React app
- `cli/` — Command-line tool for quotes
- `.github/` — GitHub configuration and instructions
